# 🏦 Bank Management System (GUI)

This is a GUI-based Bank Management System built using Python (Tkinter) and MySQL (ODBC).

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance

## Technologies Used
- Python
- Tkinter
- MySQL
- ODBC (pyodbc)

## How to Run
1. Install pyodbc:
   pip install pyodbc

2. Configure ODBC DSN

3. Run:
   python bank_gui.py

   ## Conclusion
This project demonstrates database connectivity using ODBC and GUI development using Tkinter for a real-world banking system.
